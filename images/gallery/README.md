# Gallery / Class Photos

Drop facility, class, and rolling photos here. The redesign uses Unsplash photos that match the program type until real photos arrive.

Recommended files:

- `adult-bjj-class.jpg` — adult Gi training
- `no-gi-rolling.jpg` — adult No-Gi sparring
- `kids-bjj-young.jpg` — kids 5-7 class
- `kids-bjj-older.jpg` — kids 8-12 class
- `competition-team.jpg` — competition class drill / team-photo style
- `fitness-class.jpg` — fitness / weight-loss conditioning
- `mat-overview.jpg` — wide shot of the main training room
- `belt-promotions.jpg` — belt ceremony with the team

Recommended: landscape orientation, minimum 1600px wide. Files are rendered through a `contrast(1.08) saturate(1.05)` filter — punchy phone shots work great.
