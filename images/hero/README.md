# Hero / Banner Photography

Drop hero banner shots here. These power the homepage hero and every page-header background.

Recommended files:

- `hero-home.jpg` — main homepage hero (live training, mat panorama, or oversized rolling shot)
- `hero-instructors.jpg` — page header for `/instructors`
- `hero-schedule.jpg` — page header for `/schedule`
- `hero-pricing.jpg` — page header for `/pricing`
- `hero-contact.jpg` — page header for `/contact`
- `hero-faqs.jpg`, `hero-reviews.jpg`, `hero-affiliation.jpg`, `hero-heroes.jpg`, `hero-adopt-cop.jpg`, `hero-competition.jpg`, `hero-fitness.jpg`, `hero-dress-code.jpg`

Recommended: landscape orientation, minimum 1920×1080, ideally 2400px wide. The site applies a directional dark scrim plus a slight grayscale filter — plain action photos read crisply on dark mode without further treatment.
