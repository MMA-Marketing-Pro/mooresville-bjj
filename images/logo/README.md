# Logo

Place the Mooresville BJJ logo here.

- File name: `logo.png` (or `logo.svg` / `logo.webp`)
- Format: SVG or PNG with transparent background preferred
- Min width: 400px
- Recommended: a horizontal/wordmark version sized 800×220 (or vector) — the navbar slot is 40px tall and 220px max-wide.

When you drop a real logo in here, update the `<img src="images/logo/logo.png">` reference in every HTML file (or just match this filename) and remove the `onerror="this.style.display='none'"` handler so the text fallback hides.
